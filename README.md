# itogi_1_chetverty

Написать программу, которая из имеющегося массива строк формирует 
массив из строк, длина которых меньше либо равна 3 символам

-----------------------------------------------------------------

Этап 1. Запрашиваем у пользователя длину массива и просим ввести данные.
Этап 2. Создаем массив той же длины
Этап 3. Делаем проверку введенных пользователем слов на количество символов
Этап 4. Слова, длина которых менее или равно 3 символам записываем в новый массив
Этап 5. Вывод на экран созданного массива
